# たまごっち風日記アプリ

## 概要

- キャラクターが日記の記録に反応する、たまごっち風の日記アプリ
- 日記を書くとキャラクターが喜び、書かないと寂しがる

## 主な機能

- **キャラクターシステム**
  - 3段階の表情変化（幸せ・普通・悲しい）
  - 元気度ゲージ（ハート5個で表示）
  - 状態に応じたメッセージ表示
  - 日記保存時のジャンプアニメーション

- **日記機能**
  - テキスト形式で日記を記録
  - 日付・時刻の自動記録
  - 過去の日記を時系列で表示
  - 日記件数のカウント表示

- **元気度システム**
  - 日記を書くと元気度+20%（最大100%）
  - 24時間以上書かないと元気度が徐々に低下
  - 元気度に応じてキャラクターの表情が変化

## 使い方

1. テキストエリアに今日の出来事を入力
2. 「保存する」ボタンをクリック
3. キャラクターが喜ぶ様子を確認
4. 過去の日記は下部に一覧表示

## 技術スタック

- React（Hooks使用）
- Tailwind CSS
- lucide-react（アイコン）

## デザイン
